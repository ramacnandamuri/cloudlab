# Cloud Lab

A server health diagnostics tool built while learning cloud engineering fundamentals.

## What it does
diagnose.sh checks why a server may be down by verifying:
- System ststus
- Check app logs
- Check listening ports
- Check disk usage
- Reports last 5 error messages

## Tools used
- Bash, curl, lsof, df, grep
- Git, VS Code, iTerm2

## How to run
```bash
chmod +x diagnose.sh
./diagnose.sh
```
