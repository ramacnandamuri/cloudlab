# 🤖 AIOps AI Agent

An intelligent agent that autonomously monitors 
and heals server infrastructure.

## What it does
- Monitors server health (CPU, Memory)
- Diagnoses problems intelligently
- Self-heals without human intervention
- Explains every decision it makes

## Tech Stack
- Python 3.12
- LangChain
- Ollama (Llama 3.2)
- Runs entirely locally - zero API costs!

## Run it
```bash
python -m src.agent
```
